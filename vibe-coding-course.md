# 바이브 코딩 입문 교육 과정
## 회사원을 위한 AI 여행 사진 생성기 만들기

---

## 교육 개요

| 항목 | 내용 |
|------|------|
| 대상 | 바이브 코딩 툴 경험이 거의 없는 회사원 |
| 최종 결과물 | 내 사진을 올리면 세계 여행지에 가있는 AI 사진을 만들어주는 웹앱 |
| 핵심 기술 | GitHub, Vercel, Gemini API, 바이브 코딩 툴 |

---

## 전체 교육 흐름

```
[1단계] GitHub 가입 및 기초 사용법
            ↓
[2단계] 샘플 HTML → GitHub Push
            ↓
[3단계] Vercel 가입 및 GitHub 연동
            ↓
[4단계] 배포 및 CI/CD 체험
            ↓
[5단계] 앱 기획 (바이브 코딩 기획서 생성기)
            ↓
[6단계] Gemini API 연동 - AI 여행 사진 생성기 완성
```

---

## 1단계: GitHub 가입

### GitHub란?
> 코드를 저장하고 관리하는 클라우드 저장소. 구글 드라이브와 비슷하지만 코드 전용입니다.

### 진행 방법
1. [github.com](https://github.com) 접속
2. **Sign up** 클릭
3. 이메일, 비밀번호, 사용자명 입력
4. 이메일 인증 완료
5. 무료 플랜(Free) 선택

### 확인 포인트
- [ ] GitHub 계정 로그인 성공
- [ ] 프로필 페이지 확인

---

## 2단계: 샘플 HTML → GitHub Push

### 바이브 코딩 툴로 HTML 만들기

아직 앱을 만들기 전, 간단한 샘플 HTML로 GitHub 사용법을 먼저 익힙니다.

**바이브 코딩 툴에 넣을 프롬프트:**
```
심플한 "Hello World" 스타일의 HTML 파일을 만들어줘.
- 제목: 나의 첫 번째 웹페이지
- 가운데 정렬, 깔끔한 디자인
- 파일 하나로 완성
```

### 레포지토리 생성 및 Push
1. GitHub에서 **New repository** 클릭
2. Repository name 입력: `my-travel-photo-app`
3. **Public** 선택 → **Create repository**
4. 생성된 HTML 파일 업로드 (Add file → Upload files)

### 확인 포인트
- [ ] 레포지토리 생성 완료
- [ ] HTML 파일이 레포지토리에 업로드됨

---

## 3단계: Vercel 가입 및 GitHub 연동

### Vercel이란?
> 코드를 인터넷에 무료로 올려주는 배포 서비스. GitHub와 연결하면 코드 변경 시 자동으로 반영됩니다.

### Vercel 가입
1. [vercel.com](https://vercel.com) 접속
2. **Sign Up** → **Continue with GitHub** 선택
3. GitHub 권한 허용

### Vercel ↔ GitHub 연동
1. Vercel 대시보드 → **Add New Project**
2. `my-travel-photo-app` 레포지토리 선택 → **Import**
3. 기본값 유지 → **Deploy**

### 확인 포인트
- [ ] Vercel 배포 성공 (🎉 화면)
- [ ] 배포 URL 발급 (예: `my-travel-photo-app.vercel.app`)
- [ ] 브라우저 및 모바일에서 접속 확인

---

## 4단계: CI/CD 체험

### HTML 수정 → 자동 반영 확인

**바이브 코딩 툴로 수정 요청:**
```
기존 HTML의 배경을 다크모드로 바꾸고
제목을 "AI 여행 사진 생성기"로 변경해줘
```

1. 수정된 HTML을 GitHub에 다시 업로드
2. Vercel 대시보드에서 자동 배포 시작됨 확인
3. 배포 완료 후 URL에서 변경 내용 확인

### CI/CD 개념 정리

| 용어 | 쉬운 설명 |
|------|-----------|
| CI (지속적 통합) | 코드를 올릴 때마다 자동 검증 |
| CD (지속적 배포) | 검증 후 자동으로 서비스에 반영 |

```
내가 코드 수정 → GitHub Push
        ↓  (자동)
Vercel 감지 → 빌드 → 배포
        ↓
사용자가 최신 버전 즉시 사용
```

## 5단계: 앱 기획 (바이브 코딩 기획서 생성기)

### 왜 기획서가 필요한가?
> 바이브 코딩 툴에 막연하게 "앱 만들어줘" 하면 엉뚱한 결과가 나옵니다.
> 좋은 기획서 = 좋은 프롬프트 = 좋은 결과물

### 기획서 자동 생성 방법

교육 중 직접 프롬프트를 작성하기 어려우므로,
단계별 선택지를 고르면 기획서가 자동으로 완성됩니다.

**선택 화면 흐름:**

```
Step 1. 여행지를 선택하세요
  → 파리 / 뉴욕 / 도쿄 / 발리 / 바르셀로나 / 산토리니

Step 2. 사진 분위기를 선택하세요
  → 감성적 / 모험적 / 럭셔리 / 레트로 / 자연친화적

Step 3. 계절을 선택하세요
  → 봄 / 여름 / 가을 / 겨울

Step 4. 사진 스타일을 선택하세요
  → 인스타그램 감성 / 여행 잡지 / 영화 스틸컷 / 빈티지

          [기획서 생성하기]
                ↓
        기획서 자동 완성 출력
        + 복사 버튼
```

### 생성되는 기획서 구조

```markdown
# AI 여행 사진 생성기 기획서

## 앱 개요
내 사진을 업로드하면 선택한 여행지에 있는 모습을
AI가 생성해주는 웹 애플리케이션

## 주요 기능
1. 여행지 선택 (파리, 뉴욕, 도쿄 등 6개)
2. 분위기/계절/스타일 선택
3. 내 사진 업로드
4. Gemini API로 AI 이미지 생성
5. 결과 이미지 다운로드

## 화면 구성
- 메인 화면: 단계별 선택 UI (카드 방식)
- 사진 업로드 영역
- 생성 버튼
- 결과 이미지 출력 + 다운로드 버튼

## 기술 스택
- Frontend: HTML, CSS, JavaScript (단일 파일)
- AI: Google Gemini API (이미지 생성)
- 배포: GitHub + Vercel
```

이 기획서를 바이브 코딩 툴에 붙여넣으면 → 앱 초안 완성!

## 6단계: Gemini API 연동 - AI 여행 사진 생성기 완성

### Gemini API 키 발급
1. [aistudio.google.com](https://aistudio.google.com) 접속
2. **Get API key** 클릭
3. **Create API key** → 키 복사 및 안전한 곳에 저장

> ⚠️ API 키는 절대 GitHub에 올리지 마세요!

### 바이브 코딩 툴로 앱 만들기

5단계에서 생성한 기획서를 바이브 코딩 툴에 붙여넣고 아래 내용 추가:

```
위 기획서를 바탕으로 HTML 단일 파일로 앱을 만들어줘.

- Gemini API 키 입력란을 화면 상단에 추가
- 여행지/분위기/계절/스타일은 카드 클릭 방식으로 선택
- 사진 업로드 버튼 (드래그 앤 드롭도 가능하게)
- 생성하기 버튼 클릭 시 Gemini API 호출
- 로딩 중 스피너 표시
- 결과 이미지 출력 + 다운로드 버튼
- 전체적으로 세련된 다크모드 디자인
```

### Gemini API 동작 원리

```
사용자 사진 (base64 변환)
        +
여행지/분위기/스타일 정보
        ↓
Gemini API 요청
("이 사람을 파리의 에펠탑 앞에 감성적으로 배치해줘")
        ↓
AI 이미지 생성 (약 10~20초)
        ↓
결과 이미지 출력
```

### GitHub Push → Vercel 자동 배포
1. 완성된 HTML 파일을 GitHub에 업로드
2. Vercel이 자동 감지 → 배포
3. 배포된 URL로 직접 사용 테스트
